## Python tools

Mit `py -m dir2md.dir2md` wird eine Datei inhalt.md mit Unterverzeichnissen erstellt.
In den Unterverzeichnissen werden ebenfalls inhalt.md erzeugt, auf die die übergeordnete Liste verlinkt.
 
Mit `py -m dir2md.files2md` wird eine Datei files.md mit Dateien eines Verzeichnisses erstellt.
